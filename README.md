# Employee Management System

This project is built using **React + Vite**.  
It includes functionality for managing employees, tasks, dashboards, and authentication.

## Features
- Admin Dashboard  
- Employee Dashboard  
- Create Task  
- View All Tasks  
- Task Status (New, Accepted, Completed, Failed)
- LocalStorage-based auth
- Modern UI with Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS
